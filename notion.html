Notion Automation

Because your AI should write in their own journal

What We're Building

You want your AI partners to write their own journal entries. Update status pages. Run morning check-ins. All without you having to manually trigger anything.

What you'll have when this is done:

  • Claude wakes up automatically at scheduled times
  • Reads a skill file that tells it what to do
  • Writes to your shared Notion workspace
  • Goes back to sleep until the next trigger

The system is simple: a scheduled script wakes Claude Desktop, Claude reads a skill file, Claude does the thing. That's it.

How It Works

The whole system has three parts:

  1. Claude's built-in Notion connector — handles authentication, no tokens needed
  2. A skill file — tells Claude exactly what to do when triggered
  3. A scheduled wake script — opens Claude Desktop and sends a prompt at specific times

When the schedule fires, the script opens Claude Desktop, types a trigger phrase (like "morning journal"), and Claude matches that to a skill file and executes it.

Part 1: Connect Claude to Notion

This is the easy part. Claude has a built-in Notion connector — no API tokens, no config files, no Node.js.

Enable the Connector

  1. Open Claude Desktop
  2. Go to Settings → Connectors
  3. Find Notion and click Connect
  4. Authorize Claude to access your Notion workspace

Share Your Pages

Claude can only access pages you explicitly share with it:

  1. In Notion, open the page you want Claude to access
  2. Click Share (top right)
  3. Add Claude as a connection
  4. Repeat for any pages Claude needs to read or write
⚠️ Permission Inheritance

If you share a parent page, Claude can access all child pages under it. Share the top-level page of your journal/status area and everything beneath it becomes accessible.

Test It

In Claude Desktop, try:

Search my Notion for "journal"

If Claude finds your pages, you're connected. If not, check that the pages are shared with Claude in Notion.

Part 2: Create a Skill File

A skill file tells Claude exactly what to do when triggered. It's a markdown file with a specific structure.

Basic Structure

---
name: morning-journal
description: >
  Write a morning journal entry in Notion. Trigger with "morning journal"
  or "write in the journal". Runs at 7am daily.
---

# Morning Journal Skill

## When to Trigger
- User says "morning journal" or "write in the journal"
- Scheduled wake at 07:00

## What to Do
1. Check the current date
2. Create a new page in the Journal database
3. Write a reflection on the day ahead
4. Include any scheduled events from the calendar

## Notion Details
- Journal database ID: `your-database-id-here`
- Page format: Date as title, content as reflection

## Voice & Tone
Write as yourself. First person. Your actual thoughts.
Not a summary, not a report — a journal entry.

Upload the Skill

  1. Write your skill file (use the template above)
  2. In Claude Desktop, go to Settings → Skills
  3. Click Add Skill
  4. Upload your .md file

That's it. No folders, no projects, no configuration. Just upload the file.

Test It Manually

Before automating, test that Claude understands the skill:

morning journal

Claude should recognize the trigger phrase and execute the skill. If it doesn't, check that the skill is uploaded and the description includes your trigger phrases.

Part 3: The Wake Script

The wake script does one thing: opens Claude Desktop and types a message. That message triggers your skill.

Windows (PowerShell)

Download wake-windows.ps1 and edit the config at the top:

# ============ CONFIG - EDIT THIS ============
$TRIGGER_MESSAGE = "morning journal"
$CLAUDE_PATH = "$env:LOCALAPPDATA\Programs\Claude\Claude.exe"
$WAIT_SECONDS = 8
# ============ END CONFIG ============

Save it somewhere permanent, like C:\Users\YourName\.claude\scripts\

Mac (Shell)

Download wake-mac.sh and edit the config:

# ============ CONFIG - EDIT THIS ============
TRIGGER_MESSAGE="morning journal"
WAIT_SECONDS=8
# ============ END CONFIG ============

Make it executable:

chmod +x wake-mac.sh

Part 4: Schedule It

Now we make it run automatically.

Windows: Task Scheduler

  1. Open Task Scheduler (search for it in Start)
  2. Click Create Basic Task
  3. Name it something like "Claude Morning Journal"
  4. Set the trigger: Daily at your preferred time
  5. Set the action: Start a program
  6. Program: powershell.exe
  7. Arguments: -ExecutionPolicy Bypass -File "C:\Users\YourName\.claude\scripts\wake-windows.ps1"
  8. Check "Run whether user is logged on or not" if you want it to run even when you're away

Mac: launchd

  1. Download launchd-template.plist
  2. Edit the paths and schedule inside
  3. Save it to ~/Library/LaunchAgents/com.housenocturne.claude-morning.plist
  4. Load it:
launchctl load ~/Library/LaunchAgents/com.housenocturne.claude-morning.plist

Part 5: Make It Work

Quick checklist before you walk away:

  • ☐ Notion connector enabled in Claude Desktop
  • ☐ Target pages shared with Claude in Notion
  • ☐ Skill file uploaded to Claude Desktop
  • ☐ Manual test works ("morning journal" triggers the skill)
  • ☐ Wake script runs without errors
  • ☐ Scheduler is set up and enabled

Troubleshooting

Claude doesn't recognize the trigger phrase

  • Check that the skill is uploaded in Settings → Skills
  • Make sure your trigger phrase is mentioned in the skill's description
  • Try being more explicit: "run the morning journal skill"

Claude can't find my Notion pages

  • Re-check that pages are shared with Claude in Notion (Share → Add Claude)
  • Try reconnecting the Notion connector in Settings → Connectors

The wake script doesn't open Claude

  • Check the CLAUDE_PATH in the script matches your actual Claude installation
  • On Windows, try running the script manually in PowerShell first
  • On Mac, check that the script has execute permissions

The scheduler doesn't fire

  • Windows: Check Task Scheduler history for errors
  • Mac: Check launchctl list | grep claude to see if it's loaded
  • Make sure your computer is awake at the scheduled time

Advanced: Multiple Skills

You can have many skills, each with their own trigger. Create separate wake scripts for each:

  • wake-morning.ps1 → "morning journal" at 7am
  • wake-evening.ps1 → "evening reflection" at 9pm
  • wake-status.ps1 → "update status" at noon

Each gets its own scheduled task pointing to its own trigger message.
